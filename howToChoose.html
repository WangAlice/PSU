Choosing a PSU

Things to consider:

Modular vs Semi-modular vs Non-modular

Modular Semi-modular Non-modular
What is it A modular PSU is one that comes only with ports, and wires. The user is able to use and change chords when desired. A semi-modular PSU is one that comes with a certain set of chords that are not removable, and others that are, allowing for some freedom. A non modular PSU is one that already comes with a specific set of chords. You are not able to remove or add new chords.
